Malware Classification Using CNN

Research Intern • Universiti Teknikal Malaysia Melaka (UTeM), Malaysia

Developed a Convolutional Neural Network (CNN)-based malware classification system using Python, TensorFlow, and deep learning techniques to automatically identify malware families and analyze cybersecurity threats. Utilized the Malimg and Microsoft BIG2015 benchmark datasets for model training and validation, applying data preprocessing, transfer learning, image augmentation, and hyperparameter tuning to improve classification accuracy and robustness. Built an interactive Gradio web application for real-time malware prediction and evaluated model performance using Accuracy, Precision, Recall, F1-Score, and Confusion Matrix metrics.
